# Discord Music Hub

## 概要

Discordコミュニティ向けの音楽共有Webアプリ。

ユーザーはYouTubeの曲URLを投稿し、タグ、追加理由、いいね、タイムスタンプコメントを付けられる。  
単なるプレイリストではなく、「なぜこの曲を追加したのか」「どの部分が好きなのか」を共有できるサービスにする。

## 想定ユーザー

- Discordサーバーのメンバー
- 音楽を共有したい人
- 曲に対して感想や思い出を残したい人
- タグで曲を探したい人

## 主要画面

### 1. 通常の再生・一覧画面

UI_IMAGE内の通常画面画像を参照。

構成:

- 上部ヘッダー
  - アプリ名
  - Discordサーバー選択
  - 検索バー
  - タグチップ
  - 並び替え
  - 曲追加ボタン

- 左メイン
  - 曲一覧カード
  - サムネイル
  - 再生アイコン
  - 曲名
  - アーティスト名
  - 追加者
  - タグ
  - いいね数
  - 視聴数または再生数

- 右サイドバー
  - 選択中の曲の詳細
  - 大きめのサムネイル
  - 曲名
  - アーティスト
  - 追加者
  - タグ
  - おすすめ理由
  - タイムスタンプコメント
  - YouTubeで開く
  - シェア

- 下部プレイヤー
  - 現在再生中の曲
  - 再生/停止
  - 前へ/次へ
  - 進行バー
  - 音量
  - キュー

### 2. 曲追加画面

UI_IMAGE内の曲追加画面画像を参照。

構成:

- YouTube URL入力
- URL取得後の曲プレビュー
- タグ入力
- おすすめタグ候補
- おすすめ理由入力
- タイムスタンプコメント入力
- 公開設定
  - すぐに公開
  - 下書き
- 下書き保存
- 投稿する
- 右側に投稿プレビュー
